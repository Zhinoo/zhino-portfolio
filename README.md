## üöÄ Key Features & Design Philosophy

The portfolio is built around a **Dark Neon Aesthetic** and a focus on clarity and performance.

* **Design System:** Implements a custom visual system featuring a dark base, soft gradients, and vibrant **Cyan, Purple, and Pink** neon accents for hierarchy and interactivity.
* **Performance:** Optimized for speed with efficient CSS and minimal third-party dependencies.
* **Interactivity:** Includes custom UI elements like fixed, frosted glass navigation and smooth section transitions.
* **Preloader:** Features a custom **'Cute & Soft Dot Wave'** preloader for a polished, engaging loading experience.
* **Pages Included:** Homepage (`index.html`), Project Detail Page (`project-detail-1.html`), and a Custom 404 Error Page (`404.html`).
