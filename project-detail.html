Overall Situation

This project was to create a modern and professional homepage for a new UX agency. The goal was to build a clean interface that makes people trust the company immediately.

The Challenge

The challenge was to organize a lot of information—like services, portfolio, and process—on one screen without making it look crowded or difficult to navigate.

Design Process & Solution

I implemented a Dark Glassmorphism system. This uses transparent layers and blur effects to create depth while keeping the content at the center of attention.

Key Features:

  • Modular Figma Library
  • RTL Optimized Layout
  • Soft Neon Interactions
  • Responsive Architecture
